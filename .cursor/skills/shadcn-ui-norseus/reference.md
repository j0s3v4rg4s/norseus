# shadcn/ui Reference (from ui.shadcn.com/llms.txt)

shadcn/ui is a collection of beautifully-designed, accessible components built with TypeScript, Tailwind CSS, and Radix UI primitives. Supports Next.js, Vite, Remix, Astro, React Router, and more.

## Overview

- [Introduction](https://ui.shadcn.com/docs): Open Code, Composition, Distribution, Beautiful Defaults, AI-Ready
- [CLI](https://ui.shadcn.com/docs/cli): Command-line tool for installing and managing components
- [components.json](https://ui.shadcn.com/docs/components-json): Configuration for CLI and installation
- [Theming](https://ui.shadcn.com/docs/theming): Colors, typography, design tokens
- [Changelog](https://ui.shadcn.com/docs/changelog): Release notes
